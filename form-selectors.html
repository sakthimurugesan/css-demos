:focus

h2 is used in input box in html when click on the input then we want to change the style we can use the focus

#text-1:focus{ background-color: red; width: 200px; height: 50px; color:white }

:checked

checked is used when we click on the checkbox of the input

input[name=""]:checked if we use this method only a particular group of elements have same style

input[type=""]:checked if we this method so the entire checkbox will have same style

input[name="songs"]:checked +label { color:red }

:is()

is() is used to select particular elements under same class or id

class :is(elements)

we can also select same element two different class or id

:is(class-1,class-2) element

if we want to select a group of element from different class

:is(.class-1,class-2) :is(e1,e2,e3) { style }

.div-1 :is(h1,h3,p) { color: darkgreen; }
