grid-template-row

grid-template-row is used to style rows in grid-system in css

all the property of columns can be applied for rows like repeat(),minmax(),no of rows with sizes etc..

grid-template:row/column

grid-template is a shorthand tool to style both rows and columns

col-gap|row-gap|gap

row-gap is used to mention gap between rows

col-gap is used to mention gap between column

gap is a shorthand tool to give gap for row and column

gap:row col
