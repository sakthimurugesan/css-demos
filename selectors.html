attribute selector

attribute selector is used when we define same element containing different attribute or value

example input elements type as attribute

type attribute contains different values like email,number,text,submit etc

element[attribute]{style}

input[type="text"]{background-color:red;}


input[type="number"]{background-color:blue;}


input[type="email"]{background-color:green;}


pseudo selectors

pseudo selectors are :hover,:active,:visted,:link

visited means link that we visited

link means link that we un-visited

hover means when we hover a link some style should apply

active means when we click a link some style should apply
